查看本地分支：git branch
查看远程分支：git branch -r
查看所有分支：git branch -a
查看分支关联情况：git branch -vv
将本地分支与远程分支进行关联：git branch --set-upstream-to=origin/remote_branch  local_branch
等同于：git branch -u origin/remote_branch local_branch



与远程仓库进行关联：git remote add origin 远程仓库的SSH连接
查看关联的远程仓库：git remote -v
